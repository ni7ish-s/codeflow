# Modern UX/UI Website

A modern, responsive UX/UI website built using React and Vite.  
This project focuses on clean layouts, smooth sections, and visually appealing UI components.

The project was built by following a YouTube tutorial to strengthen React fundamentals and frontend UI skills.

## Features

- Modern and minimal UI design
- Responsive layout for different screen sizes
- Reusable React components
- Smooth section-based layout
- Fast development setup using Vite

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS / UI styling libraries (as used in the project)
